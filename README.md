# react-native-keyboard-avoid-view

## Installation

```shell
npm i react-native-keyboard-avoid-view --save
```
or
```shell
yarn add react-native-keyboard-avoid-view
```

# Usage

``` js
import { KeyboardAvoidView } from 'react-native-keyboard-avoid-view'

<KeyboardAvoidView>
  <View>
    <TextInput />
    <TextInput />
    <TextInput />
    <TextInput />
  </View>
</KeyboardAvoidView>
```
